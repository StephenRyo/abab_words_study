尊敬語
抬高对方动作 · 对方↑
します
します → なさいます
社長は今何をなさっていますか。
总经理现在在做什么？
手稿例：社長 今行 なさっていますか
言います
言います → おっしゃいます
先生がおっしゃったとおりに練習しました。
我按照老师说的方法练习了。
食べます／飲みます
食べ・飲みます → 召し上がります
先生はもうお昼を召し上がりましたか。
老师您已经吃过午饭了吗？
くれます
くれます → くださいます
部長がわざわざアドバイスをくださいました。
部长特地给了我建议。
知っています
知っています → ご存じです
田中さんはこの件についてご存じでしょうか。
请问田中先生是否知晓这件事？
通用句型
お／ご＋V → ＋になる
お客様は6時の飛行機でご出発になる予定です。
客人计划乘坐6点的航班出发。
手稿注：＋になる / になります
謙譲語Ⅰ
降低己方动作·方向指向对方 · 己→对方↑
行く／来る／聞く／訪ねる
行き・来・聞きます → 伺います
明日、御社に資料を持って伺います。
明天我带着资料前往贵公司拜访。
言います（正式）
言います → 申し上げます
  • お礼
  • お詫び
  • お願い
  • ご報告
  • 申し上げる通り
心より御礼を申し上げます。
由衷地向您表示感谢。
見ます
見ます → 拝見します
送っていただいた資料を拝見しました。ありがとうございます。
我已拜读了您发来的资料，非常感谢。
会います
会います → お目にかかります
いつかまたお目にかかりたいと存じます。
希望日后能再次与您见面。
あげます
あげます → さしあげます
弊社のパンフレットをさしあげます。
我把本公司的宣传册送给您。
知っています（对人）
知っています → 存じ上げております
部長のことは存じ上げております。
我早就知道部长您了。
通用句型
お／ご＋V → ＋ある → いたす
明日の午前中にお電話いたします。
我明天上午给您打电话。
手稿：お/ご＋ある → 明日の午前中にお電話いたします
丁重語（謙譲語Ⅱ）
郑重表达己方 · 不必然指向对方
します
します → いたします
詳細については後ほどご連絡いたします。
详细内容稍后与您联系。
います
います → おります
私は東京本社に勤めております。
我目前就职于东京总公司。
行きます／来ます
行き・来ます → 参ります
これより会場に参ります。
我们现在前往会场。
言います
言います → 申します
弊社のことを簡単に申しますと……。
简单来说我们公司……
知っています（对事）
知っています → 存じております
その件については私も存じております。
关于那件事，我也知道。
⚠️ 对人用「存じ上げております」（謙譲Ⅰ）
对事用「存じております」（丁重語）
思います
思います → 存じます
このたびの件、誠に申し訳なく存じます。
对于此次之事，深感抱歉。
です（断定）
N／ナA です → でございます
応接室はあちらでございます。
会客室在那边。
あります（存在）
N があります → ございます
喫煙所は廊下のつきあたりにございます。
吸烟处在走廊尽头。
